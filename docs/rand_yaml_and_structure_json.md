# `rand.yaml` and `structure.json`

Block structure and randomization

---

## Participant to structure

During the study, participants were exposed to the blocks in a random order (counterbalancing conidtions). Which specific order of blocks each participant was given is indicated by the number assined to them in `rand.yaml`. These numbers are 0 - 7 as there are 8 total ways to randomize the order of the blocks. 

`structure.json` adds more information to the type of block and the order of the blocks based on the structure number from `rand.yaml`.
